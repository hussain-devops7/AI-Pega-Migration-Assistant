# POC Scope

## In Scope
- Upload Pega inventory file
- Store file in S3
- Analyze inventory using AI
- Generate migration plan
- Generate risk score
- Generate cost estimate
- Generate Terraform sample
- Store result in DynamoDB
- Show result on dashboard

## Out of Scope
- Real production migration
- Real DMS migration
- Real EKS deployment
- Full Pega installation
